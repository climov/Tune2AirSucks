teensy
	pin0 -> BC127 TX
	pin1 -> BC127 RX
	gnd  -> BC127 GND
	vin(5v input to teensy) -> BC127 VCC
	5v input(from 12v-to-5v reg) -> teensy vin
	3v(teensy 3.3v source) -> dock pin18
	teensy pin10 -> dock pin12
	teensy pin9 -> dock pin13

BC127
	L_OUT -> dock pin4
	R_OUT -> dock pin3
	GND(audio) -> dock pin2

pin 19/20 on ipod dock connector -> 12v input to regulator
pin 29/30 on ipod dock connector -> GND for power
